short_open_tag boolean
决定是否允许使用 PHP 代码开始标志的缩写形式（<? ?>）。如果要和 XML 结合使用 PHP，可以禁用此选项以便于嵌入使用 <?xml ?>。否则还可以通过 PHP 来输出，例如：<?php echo ‘<?xml version=”1.0”‘; ?>。如果禁用了，必须使用 PHP 代码开始标志的完整形式（<?php ?>）。
